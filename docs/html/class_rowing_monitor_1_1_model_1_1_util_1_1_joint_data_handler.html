Detailed Description

The JointDataHandler class handles the generated kinect data and provides static functions to work with the JointData.

This class implements the singleton pattern. Only one instance per application is allowed since it also counts the generated frames of the kinect sensor. The class has no public constructor.

Get an instance of the class with: JointDataHandler jdh = JointDataHandler.Instance;
